
# Used Car Sales Site

This is a JavaScript/JSX project for a used car marketplace UI. The original design is available at https://www.figma.com/design/iZ57IcwtlmmE2PMmrcVpnI/Used-Car-Sales-Site.

## Setup

1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the Vite development server.

## Scripts

- `npm run dev` — start development server
- `npm run build` — build production assets

## Notes

- The project is now using `.jsx` and `.js` source files.
- Entry point is `src/main.jsx`.
- Vite is configured with `vite.config.js`.
